Welcome to Bedrock

Bedrock is a solid foundation for contructing data analytics applications. There are three main API layers to use:

  • Dataloader API
  • Analytics API
  • Visualization API

You can find the documentation for the Bedrock core apis below.

  • Analytics
  • Visualization
